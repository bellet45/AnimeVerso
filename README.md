# ✨ Características

## 🎬 Catálogo Inteligente

* Explora miles de animes desde una interfaz moderna.
* Búsqueda rápida e instantánea.
* Filtros por:

  * Género
  * Año
  * Estado
  * Tipo
  * Ordenamiento
* Secciones de:

  * En emisión
  * Finalizados
  * Próximamente
  * Populares
  * Recién agregados

---

## 🎥 Reproductor Integrado

* Múltiples servidores de reproducción.
* Cambio de servidor con un solo clic.
* Reproducción limpia y optimizada.
* Navegación automática entre episodios.
* Interfaz inmersiva.

# ⚙ Configuración

El proyecto está preparado para funcionar tanto en desarrollo como en producción.

Incluye configuraciones para:

* Proxy de desarrollo mediante Vite.
* Reglas para Vercel.
* Reglas para Netlify.
* Gestión de CORS.
* Optimización de peticiones.

---

# 🔐 Firebase

AnimeVerso utiliza Firebase para ofrecer:

* Inicio de sesión con Google.
* Sincronización de favoritos.
* Historial.
* Preferencias del usuario.

En caso de no disponer de Firebase, la aplicación utiliza automáticamente **LocalStorage**, permitiendo seguir usando la mayoría de las funciones sin conexión con la nube.
